# Mega Scizor Team Options — 2026-04-18

## Mega Scizor — Core Identity

**Stats:** HP 70 / Atk 150 / Def 140 / SpA 65 / SpD 100 / Spe 75 (BST 600)
**Ability:** Technician (boosts moves ≤60 BP by 1.5x)
**Type:** Bug/Steel
**Item:** Scizorite (2000 VP)

**Key weakness:** 4x Fire. Fire is Scizor's existential threat — every team below addresses this differently.

**Standard Moveset (from tournament data):**
```
Bullet Punch   — 40 BP Steel, +1 priority → 60 BP after Technician (used 100%)
Bug Bite       — 60 BP Bug → 90 BP after Technician (62%)
Swords Dance   — +2 Attack (54%)
Protect / Close Combat / Dual Wingbeat / Substitute (flexible 4th)
```

Scizor's 75 base Speed is slow but not TR-optimal. It lives in "priority lane" — Bullet Punch does the work; Swords Dance makes it a late-game cleaner.

## Team 1 — Rain Balances the Fire Weakness (Meta)

**The idea:** Rain halves Fire damage, turning Scizor's 4x weakness into 2x. Simultaneously, Rain enables Swift Swim sweepers.

```
Mega Scizor @ Scizorite
Pelipper @ Focus Sash        — Drizzle auto-Rain, Hurricane
Basculegion @ Choice Scarf   — Swift Swim, Wave Crash, Shadow Ball  
Incineroar @ Chople Berry    — Intimidate pivot (further cuts physical Fire)
Politoed @ Sitrus Berry      — Backup Rain setter, Icy Wind, Helping Hand
Archaludon @ White Herb      — Electric/Dragon spread, Electroweb speed control
```

**Bring-4 gameplan:** Scizor + Basculegion frontline with Rain up — Scizor priority KOs frail targets, Scizor under +2 or Swords Dance cleans. Incineroar sits as flex pick vs hard matchups.

**Real teams:** Rod (PC215), Wolfey (PC285), and itoranotikubi (PC162) ran this in tournaments.

## Team 2 — Tailwind Hyper Offense (Slightly Off-Meta)

**The idea:** At 75 base Speed, Scizor under Tailwind hits ~150 — enough to outspeed most of the metagame unboosted. Whimsicott guarantees Tailwind via Prankster.

```
Mega Scizor @ Scizorite
Whimsicott @ Focus Sash      — Prankster Tailwind, Encore, Moonblast
Sneasler @ White Herb        — Fake Out support, Dire Claw, Close Combat
Dragapult @ Sitrus Berry     — Dragon/Ghost physical sweeper, Will-O-Wisp
Kingambit @ Black Glasses    — Supreme Overlord, Sucker Punch, Kowtow Cleave
Aerodactyl @ Choice Scarf    — Secondary Tailwind, Rock Slide, Heatwave
```

**Bring-4 gameplan:** Whimsicott + Sneasler turn 1 (Tailwind + Fake Out lock). Mega Scizor + Dragapult sweep. Kingambit closes out late game via Supreme Overlord + priority Sucker Punch.

## Team 3 — Balance with Dual Mega Flex (Off-Meta)

**The idea:** Run both Scizorite and a second Mega Stone. Each game you choose which to Mega — giving opponent a different threat to prep against in team preview.

```
Mega Scizor @ Scizorite          ┐ Choose one to Mega
Mega Gengar @ Gengarite          ┘ per game
Basculegion @ Mystic Water       — Rain sweeper
Pelipper @ Focus Sash            — Rain setter
Archaludon @ Leftovers           — Dragon/Electric bulk
Sinistcha @ Coba Berry           — TR backup, Ghost support, Tea Time
```

**Why:** Gengar-Mega threatens fast Taunt/disable strategies + has different coverage. Opponent must prep two Mega threats. Scizor cleans after Gengar opens holes. Real team by itoranotikubi (PC162).

## Team 4 — Oranguru Instruct (Very Off-Meta)

**The idea:** Oranguru's Instruct forces the ally to immediately repeat their last move. Scizor uses Bullet Punch → Oranguru uses Instruct → **Bullet Punch fires a second time in the same turn.** Double priority hits.

```
Mega Scizor @ Scizorite
Oranguru @ Sitrus Berry          — Instruct, Trick Room, Helping Hand, Protect
Farigiraf @ Mental Herb          — Armor Tail (blocks all opponent priority!), Trick Room
Incineroar @ Chople Berry        — Intimidate
Rhyperior @ Leftovers            — TR abuser, Rock Slide, Earthquake
Talonflame @ Sitrus Berry        — Gale Wings Tailwind (priority Tailwind as TR counter)
```

**Why it works:** Farigiraf's Armor Tail means no enemy Fake Out on Scizor while it sets up. Oranguru Instructs Bullet Punch after Swords Dance for devastating priority sweeping. The team is semi-TR — Farigiraf/Oranguru set TR, Rhyperior abuses, but Talonflame provides anti-TR Tailwind mode.

**Key interaction:** Earthquake from Rhyperior hits all adjacent Pokemon in doubles — use Scizor's Protect turn when Rhyperior fires spread EQ.

## Team 5 — Baton Pass Chain (Extremely Off-Meta)

**The idea:** Scizor uses Swords Dance, then Baton Passes +2 Attack into Azumarill, who then uses Belly Drum (+3 net = +5 Attack) with Huge Power. Azumarill at +5 Attack with Huge Power is a one-Pokemon win condition.

```
Mega Scizor @ Scizorite          — Swords Dance + Baton Pass
Azumarill @ Sitrus Berry         — Huge Power, Belly Drum, Aqua Jet, Play Rough
Farigiraf @ Mental Herb          — Armor Tail (Fake Out protection for Scizor's setup)
Incineroar @ Chople Berry        — Intimidate (cuts damage while Scizor sets up)
Whimsicott @ Focus Sash          — Prankster Encore (locks opponents into non-threatening moves)
Kommo-o @ Sitrus Berry           — Clanging Scales, Close Combat, Boomburst coverage
```

**Turn sequence:** Farigiraf + Mega Scizor lead → Farigiraf blocks Fake Out, Scizor uses Swords Dance → Baton Pass to Azumarill → Belly Drum → Aqua Jet priority cleanup.

## Team 6 — Grimmsnarl Screens Aggro (Off-Meta)

**The idea:** Screens (33% reduction in doubles) + Mega Scizor's already massive 140 Def makes it nearly impossible to KO physically. Set up Swords Dance freely.

```
Mega Scizor @ Scizorite
Grimmsnarl @ Lum Berry           — Prankster Reflect, Light Screen, Thunder Wave
Floette-Eternal @ ???            — Fairy Aura (boosts all Fairy moves), Light of Ruin
Sneasler @ White Herb            — Fake Out, Dire Claw, Poison Jab
Meowscarada @ Choice Scarf       — Fast Flower Trick (ignores Def), Knock Off
Kommo-o @ Sitrus Berry           — Clanging Scales + Fighting coverage
```

**Note:** Light Clay is not in Champions, so screens only last base 5 turns. Grimmsnarl still offers Prankster Thunder Wave. Floette-Eternal's Fairy Aura + Light of Ruin is one of the game's highest-WR combos (55.7% WR).

## Recommendation

- **Wins now:** Team 1 (Rain) or Team 3 (Dual Mega) — both have tournament results.
- **Creative/underexplored:** Team 4 (Oranguru Instruct) — double Bullet Punch from +2 Scizor is genuinely threatening.
- **Most fun:** Team 5 (Baton Pass) — Azumarill cleanup after a chain is extremely satisfying.
