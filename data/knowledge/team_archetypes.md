# Team Archetypes - Pokemon Champions VGC Doubles

All four weather types are simultaneously viable in Champions — a first for modern VGC. The restricted item pool (no Choice Band/Specs, Life Orb) makes weather-boosted damage and setup moves more important than raw item-boosted power.

## Rain Teams
Weather setter: Pelipper (Drizzle). Rain boosts Water moves 1.5x, weakens Fire 0.5x.
Key abusers: Basculegion (Swift Swim sweeper), Archaludon (Electro Shot needs no charge in rain).
Core pairs: Pelipper + Basculegion (55.2% WR, 21.4% usage), Archaludon + Pelipper (55.8% WR, 20.8% usage).
Thunder and Hurricane never miss in rain. Electro Shot gets free +1 SpA and fires immediately.
Strengths: Strong offensive pressure, good speed control via Swift Swim, Electro Shot + Rain is a powerful combo.
Weaknesses: Grass types (especially Venusaur), opposing weather (Tyranitar Sand Stream, Torkoal Drought), Electric types.
Sample core 6: Pelipper, Basculegion, Archaludon, Kingambit, Sneasler, Mega (flex: Mega Starmie for Huge Power water, or Mega Greninja).

## Sun Teams
Weather setter: Torkoal (Drought), Mega Charizard Y (Drought on Mega), Mega Meganium (Mega Sol sets Sun on Mega entry).
Key abusers: Venusaur (Chlorophyll doubles Speed in Sun), Mega Charizard Y (boosted Fire STAB + Solar Beam coverage).
Core pairs: Torkoal + Venusaur (56.8% WR, 6.2% usage — highest WR core), Charizard + Venusaur (55.4% WR, 12.8% usage).
Growth is now Grass-type — Venusaur gets STAB and +1 Atk/SpA in Sun (+2 SpA effectively with the Sun boost to Growth).
Strengths: Extremely high burst damage, Chlorophyll speed control, Solar Beam has no charge.
Weaknesses: Rock types (Rock Slide), opposing weather, Dragon types resist Fire.
Sample core 6: Torkoal, Venusaur, Mega Charizard Y, Incineroar, Sneasler, Garchomp.

## Sand Teams
Weather setter: Tyranitar (Sand Stream), Hippowdon (Sand Stream).
Key abuser: Excadrill (Sand Rush doubles Speed).
Core pair: Tyranitar + Excadrill (56.2% WR, 15.8% usage — second highest WR core).
Mega Excadrill gains Piercing Drill (25% damage through Protect) and massive offensive stats.
Sandstorm chips non-Rock/Ground/Steel for 1/16 HP. Rock types get +50% SpD in sand.
Strengths: Excadrill outspeeds everything in sand, Tyranitar is extremely bulky with sand SpD boost, Steel-type coverage is excellent.
Weaknesses: Water types (especially rain), Fighting types, Ground types.
Sample core 6: Tyranitar, Excadrill (or Mega Excadrill), Garchomp, Incineroar, Whimsicott, Kingambit.

## Snow/Hail Teams
Weather setter: Mega Froslass (Snow Warning on Mega), Alolan Ninetales (Snow Warning).
Key feature: Aurora Veil (halves both physical and special damage for 5 turns, requires Snow).
Slush Rush: Doubles Speed in Snow (Beartic, Alolan Sandslash).
Ice-type Defense boosted by 50% in Snow.
Strengths: Aurora Veil is extremely strong team support, defensive bulk increase is significant.
Weaknesses: Fire and Steel coverage melts Ice types, Snow doesn't boost damage directly, Fighting types.
Sample core 6: Mega Froslass (or Alolan Ninetales), Aurora Veil user, Kingambit, Garchomp, Sneasler, Incineroar.

## Trick Room Teams
Speed control: Reverses speed order for 5 turns. Slow Pokemon move first.
Key setters: Hatterene (Magic Bounce prevents Taunt), Farigiraf (Armor Tail blocks priority), Oranguru (Instruct doubles ally's last move).
Key abusers: Torkoal (Eruption in TR = devastating), Rhyperior, Ursaluna, Mega Crabominable (Iron Fist).
Archetype win rate: ~64% (highest performing archetype in Champions).
Critical note: 0 Speed IVs are impossible in Champions — all Pokemon have 31 Speed IVs. Trick Room teams cannot minimize Speed via IVs, only via negative nature and 0 SP in Speed.
Strengths: Dominates when TR is up, Eruption under TR is one of the highest-damage combos, counters speed-based teams.
Weaknesses: Taunt (unless using Hatterene), opposing Trick Room, fast setup that threatens setters, timer pressure (must win within TR turns).
Sample core 6: Hatterene, Torkoal, Farigiraf, Ursaluna, Incineroar, Mega (flex: Mega Clefable with Magic Bounce as second TR mode).

## Tailwind Hyper Offense
Speed control: Tailwind doubles team's Speed for 4 turns.
Key setter: Whimsicott (Prankster Tailwind, guaranteed to go first unless Psychic Terrain or Dark-type).
Win condition: Overwhelm the opponent with raw speed and offensive pressure before Tailwind ends.
Usage: ~20% of teams run Tailwind as primary speed control.
Strengths: Immediate speed advantage, enables slower but harder-hitting Pokemon to sweep.
Weaknesses: Fake Out pressure on Whimsicott turn 1, opposing speed control (Trick Room reverses advantage), Prankster doesn't work on Dark types.
Sample core 6: Whimsicott, Garchomp, Sneasler, Kingambit, Mega Dragonite, Incineroar.

## Balance / Goodstuffs
No single dedicated archetype — instead uses the best individual Pokemon with good synergy.
Usage: ~22% of teams (most popular style).
Core identity: Incineroar (Intimidate + Fake Out turn 1) + Sinistcha (Hospitality healing) cycling core.
Flexible game plan: Can adapt to opponent's strategy rather than forcing one win condition.
Strengths: Flexible, consistent, no single hard counter, good at adaptation.
Weaknesses: No overwhelming advantage in any dimension, can be outpaced by dedicated archetypes.
Sample core 6: Incineroar, Sinistcha, Garchomp, Sneasler, Kingambit, Mega Dragonite.

## Semi-Trick Room
Hybrid archetype that functions both inside and outside Trick Room.
Uses Pokemon with middling Speed that benefit in both modes (e.g., Incineroar at 60 base Speed).
Carries a Trick Room setter but doesn't always need to set it — can win at normal speed.
Strengths: Extremely flexible in team preview, opponent can't predict the game plan, strong in best-of-3.
Weaknesses: Master of none — neither mode is as strong as a dedicated archetype.
Sample core 6: Hatterene, Incineroar, Garchomp, Sinistcha, Torkoal, Mega Clefable.
