# Type Chart - Pokemon Champions

The type chart in Pokemon Champions is identical to Generation 9. All 18 types and their interactions are unchanged. This reference covers every offensive and defensive matchup.

## Normal Type Offensive Matchups
Normal is super effective against: nothing.
Normal is not very effective against: Rock, Steel.
Normal has no effect on: Ghost.

## Normal Type Defensive Matchups
Normal resists: nothing.
Normal is weak to: Fighting.
Normal is immune to: Ghost.

## Fire Type Offensive Matchups
Fire is super effective against: Grass, Ice, Bug, Steel.
Fire is not very effective against: Fire, Water, Rock, Dragon.
Fire has no effect on: nothing.

## Fire Type Defensive Matchups
Fire resists: Fire, Grass, Ice, Bug, Steel, Fairy.
Fire is weak to: Water, Ground, Rock.
Fire is immune to: nothing. Note: Flash Fire grants immunity to Fire-type moves.

## Water Type Offensive Matchups
Water is super effective against: Fire, Ground, Rock.
Water is not very effective against: Water, Grass, Dragon.
Water has no effect on: nothing.

## Water Type Defensive Matchups
Water resists: Fire, Water, Ice, Steel.
Water is weak to: Grass, Electric.
Water is immune to: nothing. Note: Water Absorb and Storm Drain grant immunity.

## Grass Type Offensive Matchups
Grass is super effective against: Water, Ground, Rock.
Grass is not very effective against: Fire, Grass, Poison, Flying, Bug, Dragon, Steel.
Grass has no effect on: nothing.

## Grass Type Defensive Matchups
Grass resists: Water, Grass, Electric, Ground.
Grass is weak to: Fire, Ice, Poison, Flying, Bug.
Grass is immune to: Powder/Spore moves (via type, not ability).

## Electric Type Offensive Matchups
Electric is super effective against: Water, Flying.
Electric is not very effective against: Grass, Electric, Dragon.
Electric has no effect on: Ground.

## Electric Type Defensive Matchups
Electric resists: Electric, Flying, Steel.
Electric is weak to: Ground.
Electric is immune to: nothing. Note: Volt Absorb and Lightning Rod grant immunity.

## Ice Type Offensive Matchups
Ice is super effective against: Grass, Ground, Flying, Dragon.
Ice is not very effective against: Fire, Water, Ice, Steel.
Ice has no effect on: nothing.

## Ice Type Defensive Matchups
Ice resists: Ice.
Ice is weak to: Fire, Fighting, Rock, Steel.
Ice is immune to: nothing.

## Fighting Type Offensive Matchups
Fighting is super effective against: Normal, Ice, Rock, Dark, Steel.
Fighting is not very effective against: Poison, Flying, Psychic, Bug, Fairy.
Fighting has no effect on: Ghost.

## Fighting Type Defensive Matchups
Fighting resists: Bug, Rock, Dark.
Fighting is weak to: Flying, Psychic, Fairy.
Fighting is immune to: nothing.

## Poison Type Offensive Matchups
Poison is super effective against: Grass, Fairy.
Poison is not very effective against: Poison, Ground, Rock, Ghost.
Poison has no effect on: Steel.

## Poison Type Defensive Matchups
Poison resists: Grass, Fighting, Poison, Bug, Fairy.
Poison is weak to: Ground, Psychic.
Poison is immune to: nothing.

## Ground Type Offensive Matchups
Ground is super effective against: Fire, Electric, Poison, Rock, Steel.
Ground is not very effective against: Grass, Bug.
Ground has no effect on: Flying.

## Ground Type Defensive Matchups
Ground resists: Poison, Rock.
Ground is weak to: Water, Grass, Ice.
Ground is immune to: Electric.

## Flying Type Offensive Matchups
Flying is super effective against: Grass, Fighting, Bug.
Flying is not very effective against: Electric, Rock, Steel.
Flying has no effect on: nothing.

## Flying Type Defensive Matchups
Flying resists: Grass, Fighting, Bug.
Flying is weak to: Electric, Ice, Rock.
Flying is immune to: Ground.

## Psychic Type Offensive Matchups
Psychic is super effective against: Fighting, Poison.
Psychic is not very effective against: Psychic, Steel.
Psychic has no effect on: Dark.

## Psychic Type Defensive Matchups
Psychic resists: Fighting, Psychic.
Psychic is weak to: Bug, Ghost, Dark.
Psychic is immune to: nothing.

## Bug Type Offensive Matchups
Bug is super effective against: Grass, Psychic, Dark.
Bug is not very effective against: Fire, Fighting, Poison, Flying, Ghost, Steel, Fairy.
Bug has no effect on: nothing.

## Bug Type Defensive Matchups
Bug resists: Grass, Fighting, Ground.
Bug is weak to: Fire, Flying, Rock.
Bug is immune to: nothing.

## Rock Type Offensive Matchups
Rock is super effective against: Fire, Ice, Flying, Bug.
Rock is not very effective against: Fighting, Ground, Steel.
Rock has no effect on: nothing.

## Rock Type Defensive Matchups
Rock resists: Normal, Fire, Poison, Flying.
Rock is weak to: Water, Grass, Fighting, Ground, Steel.
Rock is immune to: nothing. Note: Rock types gain +50% SpD in Sandstorm.

## Ghost Type Offensive Matchups
Ghost is super effective against: Psychic, Ghost.
Ghost is not very effective against: Dark.
Ghost has no effect on: Normal.

## Ghost Type Defensive Matchups
Ghost resists: Poison, Bug.
Ghost is weak to: Ghost, Dark.
Ghost is immune to: Normal, Fighting.

## Dragon Type Offensive Matchups
Dragon is super effective against: Dragon.
Dragon is not very effective against: Steel.
Dragon has no effect on: Fairy.

## Dragon Type Defensive Matchups
Dragon resists: Fire, Water, Grass, Electric.
Dragon is weak to: Ice, Dragon, Fairy.
Dragon is immune to: nothing.

## Dark Type Offensive Matchups
Dark is super effective against: Psychic, Ghost.
Dark is not very effective against: Fighting, Dark, Fairy.
Dark has no effect on: nothing.

## Dark Type Defensive Matchups
Dark resists: Ghost, Dark.
Dark is weak to: Fighting, Bug, Fairy.
Dark is immune to: Psychic. Note: Prankster-boosted status moves fail against Dark types.

## Steel Type Offensive Matchups
Steel is super effective against: Ice, Rock, Fairy.
Steel is not very effective against: Fire, Water, Electric, Steel.
Steel has no effect on: nothing.

## Steel Type Defensive Matchups
Steel resists: Normal, Grass, Ice, Flying, Psychic, Bug, Rock, Dragon, Steel, Fairy.
Steel is weak to: Fire, Fighting, Ground.
Steel is immune to: Poison. Note: Steel types are not affected by Sandstorm chip damage.

## Fairy Type Offensive Matchups
Fairy is super effective against: Fighting, Dragon, Dark.
Fairy is not very effective against: Fire, Poison, Steel.
Fairy has no effect on: nothing.

## Fairy Type Defensive Matchups
Fairy resists: Fighting, Bug, Dark.
Fairy is weak to: Poison, Steel.
Fairy is immune to: Dragon.
