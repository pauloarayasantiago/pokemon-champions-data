# Speed Tiers - Pokemon Champions Level 50

All Pokemon have 31 Speed IVs (locked). Speed at Lv50 is determined by base speed, nature (1.1x or 0.9x), and Stat Points (0-32 SP). There is NO way to reduce Speed via IVs — this is critical for Trick Room teams.

## Speed Calculation
At Lv50 with 31 IVs: Speed = floor(((2 * Base + 31) * 50 / 100 + 5) * Nature)
- Max Speed (positive nature, 32 SP): Base stat + nature bonus + full SP investment
- Min Speed (negative nature, 0 SP): Base stat with 0.9x nature, 0 SP

## Fastest Unboosted Pokemon (Max Speed, Positive Nature)
- Mega Alakazam: ~202 Speed (fastest in format)
- Mega Aerodactyl: ~200 Speed
- Mega Gengar: ~178 Speed
- Dragapult: ~213 Speed (if available; 142 base)
- Sneasler: ~152 base, ~167 max; Unburden doubles to ~334 after item consumed
- Greninja: ~144 base, ~158 max
- Garchomp: ~102 base, ~154 max (Jolly)
- Mega Dragonite: 80 base (slower than base Dragonite at 80)

## Key Speed Benchmarks
These are the speeds that matter most for offensive and defensive calcs:
- 102: Garchomp base speed — THE defining speed tier. Outspeeding Garchomp is critical.
- 80: Dragonite base (and Mega). Often needs Dragon Dance or Tailwind.
- 60: Incineroar base — very slow, often outsped. Good in Trick Room.
- 150: Sneasler base — one of the fastest non-Mega Pokemon.
- 65: Hatterene base — Trick Room setter, wants to be slow.
- 70: Torkoal base — slow for Eruption in Trick Room.
- 130: Mega Gengar (from 110 base Gengar).
- 115: Mega Charizard Y (same as base Charizard).
- 100: Mega Meganium, Mega Feraligatr.

## Trick Room Speed Tiers (Slowest at Lv50)
In Trick Room, slower Pokemon move first. Since 0 Speed IVs are impossible in Champions, the minimum Speed is higher than in S/V. Min Speed = negative nature, 0 SP.
- Torkoal: 70 base → ~58 min speed (good TR abuser)
- Rhyperior: 40 base → ~40 min speed (one of slowest, excellent TR)
- Hatterene: 65 base → ~54 min speed (TR setter that's slow enough to benefit)
- Ursaluna: 50 base → ~45 min speed (strong TR abuser)
- Oranguru: 60 base → ~49 min speed (Instruct + TR setter)
- Farigiraf: 60 base → ~49 min speed (Armor Tail TR setter)
- Mega Crabominable: 43 base → ~41 min speed (Iron Fist, very slow)
- Incineroar: 60 base → ~49 min speed (functions well in both modes)

## Speed After Tailwind (x2)
Tailwind doubles the team's Speed for 4 turns. Key thresholds after Tailwind:
- Garchomp (Jolly max): 154 → 308 (outspeeds everything unboosted)
- Kingambit (Adamant max): 112 → 224
- Incineroar (max neutral): 112 → 224
- Basculegion (Adamant max): 130 → 260
- Dragapult doesn't usually need Tailwind — already fastest tier

## Speed After Weather Abilities (x2)
Weather speed-doubling abilities:
- Swift Swim (Rain): Basculegion 130 → 260, Kingdra 85 → 170
- Chlorophyll (Sun): Venusaur 80 → 160, Victreebel 70 → 140
- Sand Rush (Sand): Excadrill 88 → 176, Mega Excadrill → even higher
- Slush Rush (Snow): Beartic 50 → 100, Alolan Sandslash 65 → 130

## Speed Control Moves Impact
- Tailwind: Doubles team speed for 4 turns
- Trick Room: Reverses speed order for 5 turns (-7 priority to set)
- Icy Wind: -1 Speed to both opponents (spread, 55 BP Ice)
- Electroweb: -1 Speed to both opponents (spread, 55 BP Electric)
- Thunder Wave: Halves target Speed (paralysis), but only 12.5% full para in Champions
- Sticky Web: -1 Speed on grounded switch-in (hazard, rare in Doubles)
- Bulldoze: -1 Speed to all adjacent (spread, 60 BP Ground, hits ally too)
- String Shot: -2 Speed to target

## Unburden Interactions
Sneasler with Unburden: Speed doubles after item is consumed.
- White Herb: Consumed on Intimidate → Unburden activates → ~334 Speed
- Focus Sash: Consumed after surviving a hit → Unburden activates
- Psychic Seed: Consumed on Psychic Terrain entry → Unburden + SpD boost
This makes Sneasler one of the fastest Pokemon in the format after turn 1.

## Speed Ties and Priority
When Pokemon have equal Speed, the game randomly decides who moves first (50/50).
Priority moves bypass Speed entirely:
- +4: Protect (8 PP in Champions)
- +3: Fake Out (turn 1 only!), Quick Attack, Mach Punch, Aqua Jet, Bullet Punch, Ice Shard, Shadow Sneak, Accelerock, Sucker Punch (conditional), Extreme Speed (+2)
- +1: Prankster-boosted status moves (fails vs Dark types)
- 0: Most moves
- -7: Trick Room
